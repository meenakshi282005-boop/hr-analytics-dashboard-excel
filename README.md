# HR Analytics Dashboard (Excel)

## Overview
This project is an interactive HR Analytics Dashboard built using Microsoft Excel. The dashboard helps analyze employee attrition, income trends, workforce distribution, and key HR metrics through visualizations and KPIs.

## Key Metrics
- Total Employees: 1470
- Attrition Count: 237
- Attrition Rate: 16.12%
- Average Monthly Income: $6,503
- Average Age: 37

## Key Insights
- Research & Development department has the highest attrition (133 employees).
- Male employee attrition is higher than female employee attrition.
- Laboratory Technicians show the highest attrition among job roles.
- Sales department has the highest average monthly income.
- Life Sciences has the highest number of employees among education fields.
